# Encryption Tool

图形化加密工具

## 功能

- 密钥管理
- 文本加密
- 文件加密

## 支持算法

| 类型 | 算法 |
|------|------|
| 对称加密 | AES-256、ChaCha20 |
| 非对称加密 | RSA-4096、X25519 |
| 后量子加密 | ML-KEM-768 |
